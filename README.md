# Hospital Readmission Prediction Using Machine Learning

## Project Overview

This project predicts whether a diabetic patient is likely to be readmitted within 30 days using machine learning models.

## Dataset

- Diabetes 130-US Hospitals for Years 1999–2008
- Source: Kaggle

## Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- XGBoost

## Models Used

- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

## Best Model

Random Forest

Accuracy: **82.81%**
